# Template Manager Overview

The Template Manager allows you to create, edit, and assign employee templates for payroll processing. Templates define allowances, deductions, wage types, and other compensation settings.

---

## Creating a New Template

To create a new template, click **Template Manager** in the side menu, then click the **New Template** button. Fill out the required fields: Template Name, Description, Type, and GL Account.

---

## Finding and Editing a Template

To edit a template, click **Template Manager** in the side menu. Search for the desired template using the search bar and click on it from the results. Click the **Edit** button in the top right. You can edit the Description, Template Type, and GL Account.

---

## Allowance Template

An Allowance Template defines supplementary payments to employees. When editing an allowance template, configure the following fields:

- **Type**: Select the allowance type (Car, Travelling, etc.)
- **Taxable**: Enable or disable tax calculation for this allowance
- **Percentage Base**: Enable to calculate the allowance based on a percentage

---

## Deduction Template

A Deduction Template defines amounts or percentages to subtract from employee compensation. When editing a deduction template, you can configure:

- **Statutory Deduction**: Enable or disable predefined statutory deduction rates
- **Percentage Deduction**: Enable percentage-based deductions that can be entered by employer or employee

---

## Fixed Wage Template

A Fixed Wage Template defines consistent salary or wage amounts. When editing a fixed wage template, you can adjust the **Credit Amount** field to set the fixed wage value.

---

## Variable Wage Template

A Variable Wage Template defines compensation rates for different work hour types. When editing a variable wage template, configure these rate fields:

- **NT Rate**: Regular/normal time rate
- **OT Rate**: Overtime rate
- **DT Rate**: Double time rate
- **TT Rate**: Triple time rate
- **Default Units**: Default unit value for wage calculations

---

## Vacation Payable Template

Vacation Payable settings are automatically managed by the payroll system. These settings control how vacation time accrual and payouts are calculated.

---

## Severance Template

Severance Template configuration is not yet available in the current system version.

---

## Loans Template

Loans Template configuration is not yet available in the current system version.

---

## Assigning Employees to a Template

To add an employee to a template, click **Template Manager** in the side menu and search for the desired template using the search bar. Click on the template from the results and navigate to the **Employee** tab. Click the **Add Employee** button, search for the employee by first name, last name, or employee ID, select the required fields, and confirm to add the employee.
